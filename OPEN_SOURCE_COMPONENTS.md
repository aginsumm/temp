# 非遗数字生命互动引擎 - 开源代码与组件使用说明

**前端**：基于 React + TypeScript 构建类型安全的应用，采用 Ant Design 提供统一 UI 风格，使用 ECharts 和 Three.js 实现知识图谱与数据可视化，通过 Zustand 进行轻量级状态管理，Framer Motion 驱动流畅动画效果。

**后端**：FastAPI 提供高性能 RESTful API，Pydantic 确保数据验证与类型安全，SQLAlchemy ORM 简化数据库操作，AsyncPG/AioSQLite 支持异步数据库访问，Redis 提供缓存加速，Neo4j 和 PyMilvus 实现知识图谱与向量检索，Celery 处理异步任务队列，DashScope 接入大语言模型能力。

**许可证**：全部组件采用 MIT、Apache-2.0、BSD、ISC 等宽松许可证，允许商业使用、修改和分发，仅需保留原始版权声明。
