#### 非遗类别

传统技艺

#### 正文

**一、基本简介（约180字）**
 成都银花丝是成都地区最具地方特色的金银细工技艺之一，以白银为主要材料，通过搓丝、垒丝、填丝、焊接等手工方法制成各类器物和饰品。其工艺以“花丝平填”最具代表性，造型细密精巧，整体风格典雅华美，兼具民族特色与地方工艺风格。传统银花丝产品包括瓶、盘、薰、鼎、盒以及钗、环、镯等摆件和饰品，近现代又发展出银丝画等新品类。作为成都传统金工艺术的重要代表，成都银花丝兼具工艺价值、审美价值和收藏价值。

**二、历史渊源（约290字）**
 根据中国非遗资料，成都金银器制作在秦汉时代就已较为发达，至清代中叶已形成一定规模，其中银花丝制作技艺尤为突出，已有两千多年的历史。它并不是凭空出现的新工艺，而是在成都地区长期金银器制作传统基础上发展出来的高难度细工技艺。随着地方工艺审美和手工业分工的成熟，成都银花丝逐渐形成以细银丝组织装饰纹样、强调平填效果和手工精度的独特技术路线。其长期存在说明成都不仅是丝织、刺绣重镇，也在贵金属工艺方面形成了精细而独特的地方传统。银花丝发源于成都、兴盛于成都，是成都地方工艺谱系中不可忽视的一支。

**三、核心技艺 / 表演形式 / 工艺特点（约470字）**
 成都银花丝的核心工艺在于“以丝成形、以填见巧”。它以白银为原料，先将银材加工成不同粗细的银丝，再根据设计图稿制作边框、填充纹样，并通过焊接、编织等手段完成整体造型。中国非遗资料提到，它综合运用花丝和錾刻、填丝、垒丝、穿丝、搓丝、焊接等多种技艺，按照设计要求制作手工艺品。换句话说，银花丝不是单一步骤，而是一整套高度精细的贵金属加工系统。

其中最有代表性的工艺特点是“平填”。所谓平填，是在轮廓之内用银丝进行细密填充和组织，使图案在平面上呈现出纹理繁复、层次细腻的效果。相关报道还指出，成都银花丝的重要特征是“无胎成形”，即在没有明显依附胎体的情况下，通过银丝本身的结构完成图案与装饰塑造，这对匠人的手工控制能力要求极高。

其工艺特点可概括为：用材贵重、丝工精细、平填见长、造型别致、装饰纹理繁密。传统产品包括摆件和首饰，现代又衍生出银丝画等新形式，并尝试结合浮雕、镶宝、烧蓝、珐琅等元素进行创新。正因工艺复杂、制作周期长、学习成本高，成都银花丝既显得珍贵，也面临传承难度大的现实问题。

**四、代表作品 / 代表场景 / 代表器物（约220字）**
 成都银花丝的代表产品主要包括瓶、盘、薰、鼎、盒等传统陈设器，以及钗、环、镯等佩饰类作品；近年还发展出“银丝画”等新品种。其传统场景包括手工艺作坊制作、礼品陈设、室内装饰与佩戴使用。代表器物和工具则包括纯银材料、拉丝与搓丝工具、焊接工具、錾刻工具以及用于图稿设计和纹样填充的工艺辅助器具。对成都而言，银花丝最能体现的是传统金工中“细、巧、密、雅”的工艺追求。

**五、传承人 / 保护现状 / 当代传播（约230字）**
 成都银花丝制作技艺的国家级代表性传承人为道安。公开报道提到，她是目前少数能够完整掌握成都银花丝全部流程的代表性艺人之一，这也侧面说明该技艺传承难度较高。由于银花丝制作工序复杂、训练周期长、市场规模有限，其保护压力长期存在。与此同时，传承人及其团队也在尝试工艺创新，例如将浮雕、镶宝、烧蓝、珐琅等元素融入银花丝制作，并开发更符合现代审美的产品类型，以增强当代传播与市场转化能力。

**六、地域文化关联说明（约180字）**
 成都银花丝与成都地域文化的关系，主要体现在其“发源于成都、兴盛于成都”的地方工艺属性上。它与蜀锦、蜀绣一样，都体现了成都传统手工业高度精细、重视装饰和审美品位的城市文化特征。若说蜀锦代表成都的丝织文明，蜀绣代表成都的针线美术，那么银花丝则代表成都在贵金属细工领域的独特创造力，共同构成了成都非遗工艺版图中的重要部分。
