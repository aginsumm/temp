#### 非遗类别

传统技艺

#### 正文

**一、基本简介（约180字）**
 蜀锦是指四川成都地区生产的丝织提花织锦，是中国最负盛名的传统织锦之一，也是成都最具标志性的手工技艺之一。它历史悠久，工艺精密，纹样华丽，长期被视为高等级织物和重要贸易品。蜀锦并不只是一般意义上的丝织品，而是依靠复杂织机、纹样设计和多工序配合完成的高级织造工艺。成都因盛产蜀锦而获得“锦城”“锦官城”等称谓，锦江之名也与蜀锦洗濯活动相关。蜀锦因此不仅是工艺品，更是成都城市身份的重要文化象征。

**二、历史渊源（约290字）**
 蜀锦手工织造技艺在成都地区已延续两千多年。中国非遗资料指出，蜀锦最早的文字记载可见于春秋时期秦惠文王年间，而到战国时期，蜀锦已成为重要贸易品。它兴于秦汉，盛于唐宋，明末衰落，清代中晚期有所恢复，近代又一度陷入危机。蜀锦不仅在地方经济中占有重要地位，也曾沿着丝绸之路进入更广阔的流通和交流网络。早在秦汉三国时期，它就已走出四川，进入少数民族地区，在经济往来、文化传播和民族交流中发挥过重要作用。因此，蜀锦的历史并不局限于一项手工技艺本身，而是与成都在中国丝织文明中的地位紧密相连。

**三、核心技艺 / 表演形式 / 工艺特点（约470字）**
 蜀锦的核心在于织造技艺，而非单纯材料。其传统技术体系以花楼织机为中心，从纹样设计、挑花结本到挽花工与织工协作生产，形成了一套分工明确、流程复杂的手工织锦系统。也就是说，蜀锦并不是“把丝线织成布”这么简单，而是要先完成图案结构设定，再通过复杂的经纬组织和人工配合，将图样转化为织物表面的色彩与纹理。

花楼织机是蜀锦技艺的关键，它要求不同工种相互配合，既考验技术经验，也考验对图样和织造逻辑的理解。其后出现的有梭机械织机，技术原理与传统织造一脉相承，但传统花楼织机更能体现蜀锦手工艺的历史原貌。蜀锦成品通常具有纹样工整、色彩浓丽、组织严密、质地华美等特点，既具装饰价值，也具有制度、礼仪和贸易层面的文化含义。

其工艺特点可概括为：历史延续性强、织机结构复杂、纹样设计讲究、人工协作程度高、成品装饰性强。由于现代工业织物替代和手工技艺萎缩，传统蜀锦曾面临明显衰退风险。也正因如此，蜀锦织造技艺被纳入国家级和联合国层面的非遗保护体系，成为中国传统丝织文明中极具代表性的活态遗产。

**四、代表作品 / 代表场景 / 代表器物（约220字）**
 蜀锦传统上多用于礼仪服饰、陈设织物、装饰面料及高等级丝织用品，其代表器物主要围绕花楼织机、结本工具、丝线材料与纹样设计样本展开。典型场景包括织造作坊、宫廷和礼仪用途、贸易流通以及现代博物馆展示空间。蜀锦作为织物本身，代表性不一定体现为某一件单件作品，而更多体现在典型纹样、传统织机和织造工序的整体系统之中。对于成都而言，蜀锦最有代表性的“作品”之一，其实就是“锦城”这一城市文化记忆本身。

**五、传承人 / 保护现状 / 当代传播（约230字）**
 蜀锦当前保护的关键问题，在于传统手工织机被现代织机替代，手工技艺因此出现萎缩和衰退趋势。中国非遗资料明确提到，采取切实措施确保蜀锦传承已是当务之急。近年来，相关社区、群体和个人持续对传统机具和技法进行研究改进，并搭建发展平台，推动匠人培养和活态传承。随着博物馆展示、非遗节活动、研学体验和文创开发的发展，蜀锦已重新进入公众视野，从历史性高端织物转化为兼具教育、审美和文化传播价值的重要非遗资源。

**六、地域文化关联说明（约180字）**
 蜀锦与成都的地域联系极深。成都“锦城”“锦官城”的别称直接来源于蜀锦生产历史，而“锦江”名称也与蜀锦洗濯传统有关。换言之，蜀锦不是成都众多工艺中的一种而已，而是直接嵌入城市命名、城市记忆与地方历史叙事之中的核心文化符号。它植根于蜀文化，见证了成都作为丝织重镇、商贸城市和文化交流节点的长期发展过程。
