#### 非遗类别

传统戏剧。川剧为国家级非物质文化遗产代表性项目。

#### 正文

**一、基本简介（约180字）**
 川剧是流行于四川、重庆及云南、贵州、湖北部分地区的地方戏曲剧种，也是中国西南地区影响力最大的地方戏之一。它兼具文学、音乐与综合表演艺术特征，以唱、念、做、打并重，舞台表现力极强。川剧最鲜明的特点，是由高腔、胡琴、弹戏、昆曲、灯调五种声腔共同构成的表演体系，因此常被概括为“五腔共和”。它既保留了中国古老戏曲声腔演变的历史痕迹，又形成了鲜明的巴蜀地方审美与生活气息，是成都城市文化和四川戏曲传统的重要代表。

**二、历史渊源（约290字）**
 川剧并非单一起源，而是在明末清初以后多种外来声腔不断传入四川，并与本地民间音乐长期融合的结果。昆曲、弋阳腔、青阳腔、陕西梆子、湖北汉调、徽调等声腔先后进入四川，在乾隆、嘉庆年间，又与薅秧调、川江号子、地方小调及宗教音乐逐步结合，完成了外来戏曲声腔的“四川化”过程。到辛亥革命前后，高腔、昆曲、胡琴、弹戏以及四川本土灯戏在同台演出中进一步融汇，最终形成川剧这一成熟剧种。正因其由多种声腔整合而成，川剧被视为中国戏曲声腔流变史的重要“活化石”，具有较高的艺术史和地方文化史价值。

**三、核心技艺 / 表演形式 / 工艺特点（约470字）**
 川剧的核心，并不只是大众熟悉的“变脸”，而是一整套成熟而复杂的戏曲表演系统。首先在音乐层面，它由高腔、胡琴、弹戏、昆曲、灯调五种声腔组成，不同剧目、人物类型与情绪表达，会调动不同的声腔资源，形成丰富的音乐层次。其次在表演层面，川剧遵循中国戏曲“唱、念、做、打”的基本规律，但在念白节奏、帮腔形式、锣鼓配合、身段动作和武戏处理上，都发展出明显的地方特色。

川剧高腔中的帮腔尤其具有代表性，它强化了戏剧氛围与音乐张力；而在舞台技巧上，变脸、吐火、藏刀等绝技使川剧具有极强的观赏性和辨识度。不过，这些绝技只是川剧表演中的一个部分，真正支撑川剧艺术的是完整的角色行当体系、数千个传统与新编剧目、成熟的唱腔结构和世代积累的表演程式。

其工艺特点可概括为：声腔体系复合、音乐表现层次丰富、念白地方色彩浓厚、舞台程式鲜明、绝技表演突出、文戏武戏并重。传统代表剧目与现代整理改编剧目并存，也说明川剧并非静态遗产，而是一种持续生长、能够与当代观众建立联系的活态戏曲艺术。

**四、代表作品 / 代表场景 / 代表器物（约220字）**
 川剧代表剧目十分丰富，传统剧目和创作剧目共六千余个，常被提及的有《黄袍记》《九龙柱》《幽闺记》《春秋配》《五子告母》等；新中国成立后整理改编的《柳荫记》《白蛇传》《拉郎配》，以及改革开放后影响较大的《巴山秀才》《变脸》《死水微澜》等，也都具有代表性。川剧最典型的使用场景是剧场演出、戏园子、节庆展演和地方文化活动。代表器物则包括戏服、脸谱、盔帽、折扇、刀枪把子和锣鼓乐器等。

**五、传承人 / 保护现状 / 当代传播（约230字）**
 川剧已形成较完整的保护和传承体系，国家级代表性传承人中有晓艇、陈巧茹、陈智林等。当前川剧的主要任务，一方面是维持传统剧目、声腔和行当程式的延续，另一方面是吸引年轻观众、拓展当代传播空间。近年来，川剧借助剧场演出、校园传播、文旅演艺和短视频传播等方式提升社会可见度，其中“变脸”等舞台技艺尤易形成公众认知入口。但从整体上看，真正艰难的部分仍在于完整戏曲训练体系和经典剧目演出生态的持续维持。

**六、地域文化关联说明（约180字）**
 川剧与成都地域文化关系极为密切。成都长期是巴蜀地区文化、商业与演艺活动的中心，茶馆文化、市民娱乐传统和戏园生态为川剧发展提供了深厚土壤。川剧中的语言节奏、生活气息、幽默方式和人物塑造，都带有鲜明的成都平原与巴蜀社会特色。可以说，川剧不仅是一种戏曲剧种，也是成都地方性格、城市生活方式和区域审美的重要表达。
