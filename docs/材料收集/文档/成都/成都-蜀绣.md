#### 非遗类别

传统美术

#### 正文

**一、基本简介（约180字）**
 蜀绣又称川绣，是起源于川西民间、长期流传于成都及周边地区的重要刺绣艺术，也是中国四大名绣之一。它以丝线和彩缎为主要材料，在长期发展中形成了严谨细腻、光亮平整、构图疏朗、浑厚圆润、色彩明快的艺术风格。蜀绣既擅长表现花鸟虫鱼等细腻题材，也能表现山水、人物等较具气势和层次的图像。作为成都代表性传统手工艺，蜀绣兼具实用装饰、审美欣赏与民间工艺价值，是巴蜀丝织文化与女性手工传统的重要载体。

**二、历史渊源（约290字）**
 蜀绣的发展与成都平原发达的蚕桑丝织业密切相关。古代川西平原盛产蚕桑，“蜀”本身就与丝织经济有深厚联系。《史记》记载，春秋时期蜀地已将丝织品、麻织品运往秦国贸易；至两晋时期，刺绣产品已成为蜀地特产。汉末三国时期，蜀绣产品已受到官府控制，说明其生产和价值已相当成熟。隋唐以后，随着丝绸之路贸易往来活跃，织绣品需求增长，蜀绣进入快速发展阶段，并达到历史高峰。明清时期，除闺阁女红外，四川地区出现了越来越多专业刺绣人员和作坊。到20世纪上半叶，成都已形成较有规模的蜀绣行业，显示出其从家庭手艺向专业工艺体系发展的过程。

**三、核心技艺 / 表演形式 / 工艺特点（约470字）**
 蜀绣的核心在于针法体系。根据中国非遗资料，蜀绣以针法见长，共有12大类、122种针法，这意味着它并不是单靠材料华丽取胜，而是依赖严密的手工技法体系来完成画面塑造。常见的技术逻辑，是根据不同题材、肌理和明暗要求，灵活组合多种针法，使线迹方向、疏密、光感和层次彼此配合，从而形成丰富的视觉效果。

蜀绣以本地织造的红、绿等色缎和散线为主要原料，用线工整稳重，施针严谨。它尤其善于处理花卉、禽鸟、虫鱼等具有细节变化的图像，也能通过线色交错和针脚组织表现山水的宏阔气势。受四川地理环境、地方风俗和文化艺术的综合影响，蜀绣逐渐形成了严谨细腻、光亮平整、构图疏朗、浑厚圆润、色彩明快的独特风格。

其工艺特点可概括为：针法系统成熟、造型表现细腻、设色典雅而不浮艳、画面平整而有光泽、既有装饰性也有写实能力。近现代以来，蜀绣除传统服饰、被面、枕套、屏风等用途外，也不断进入收藏、展览、礼品和文创领域。它既保留着传统女红和手工美术的属性，也在当代消费与展示环境中获得新的传播空间。

**四、代表作品 / 代表场景 / 代表器物（约220字）**
 蜀绣常见作品包括屏风、挂件、服饰绣片、被面、枕套、桌围、镜框绣和装饰摆件等。题材上以花鸟、熊猫、鱼虫、山水、人物最具代表性，其中既有民俗装饰作品，也有适于陈设和收藏的精品绣作。其传统使用场景包括嫁妆制作、日常织绣用品、室内陈设和节庆礼品。代表器物和工具主要有绣绷、针、丝线、缎料、剪刀、描样工具等。对成都而言，蜀绣既是手工艺，也是地方女性劳动传统与丝织文明的重要象征。

**五、传承人 / 保护现状 / 当代传播（约230字）**
 蜀绣当前面临的主要问题，是手工制作耗时长、投入产出比偏低，加之家庭作坊式传承局限，容易出现后继乏人和技艺流失风险。中国非遗资料明确提到，蜀绣曾一度面临濒危状态，急需保护。与此同时，随着社会审美需求变化和文化产业发展，蜀绣传承人也在不断创造新针法、拓展新题材，并将作品推向更广泛的市场。今天的蜀绣，已经不只停留在传统家用织绣层面，还通过博物馆展览、非遗工坊、文创产品和旅游展示等方式进入新的传播场景。

**六、地域文化关联说明（约180字）**
 蜀绣与成都地域文化的联系建立在“蜀地蚕桑—丝织经济—手工美术”这条清晰链条之上。成都平原自古丝织发达，为蜀绣提供了材料基础、市场需求与审美土壤。蜀绣风格中的细腻、明快、浑厚和生活气息，也与巴蜀地区的自然环境和地方文化密切相关。它不仅代表成都传统工艺的精致程度，也反映了成都作为“天府之国”历史上物产丰饶、生活讲究的城市文化特征。
