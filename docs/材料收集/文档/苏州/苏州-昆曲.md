#### 非遗类别

传统戏剧

#### 正文

**一、基本简介（约180字）**
 昆曲，又称昆腔、昆剧，发源于江苏苏州昆山一带，是中国现存最古老、最成熟的戏曲声腔系统之一，被誉为“百戏之祖”。它以曲辞典雅、唱腔婉转、表演细腻著称，集文学、音乐、表演、舞蹈、美术于一体，对后世京剧、川剧等诸多剧种都产生了深远影响。2001年，昆曲入选联合国教科文组织首批“人类口头和非物质遗产代表作”，后转入人类非物质文化遗产代表作名录；2006年列入第一批国家级非物质文化遗产名录。对苏州而言，昆曲不仅是地方文化标识，也是中国古典戏曲艺术高度成熟的代表。 

**二、历史渊源（约290字）**
 昆曲的源头可追溯到元末明初南戏在昆山一带的发展。早期由顾坚等人推动昆山腔形成雏形，至明代嘉靖年间，魏良辅吸收北曲及其他声腔之长，对原有南曲体系进行系统改革，逐渐塑造出婉转流丽、节奏舒缓、讲究字音腔格的“水磨调”风格，昆曲由此基本成型。随后，梁辰鱼创作传奇《浣纱记》，将原本偏重清唱的昆山腔推进到成熟舞台戏剧形态，标志着昆剧真正诞生。明清时期，昆曲沿运河北上南下、向全国传播，成为全国性剧种，并在传奇文学、舞台程式、角色体系与审美趣味上形成完整传统。清中叶后，昆曲影响力有所减弱，但其艺术规范始终作为中国戏曲的重要根基保留下来。新中国成立后，苏州及相关地区持续开展抢救、研究、建院、立法与推广，推动昆曲进入新的保护与复兴阶段。 

**三、核心技艺 / 表演形式 / 工艺特点（约470字）**
 昆曲的核心特征，首先体现在唱腔系统上。其声腔讲求“水磨调”式的圆润、婉转、绵密，强调吐字行腔的精致统一，既要求字音清晰，又要求旋律流动自然，因此演唱中对气息、口法、节奏、字头字腹字尾的处理极为严格。昆曲的文学性也非常强，很多经典剧目本身就是中国古典戏曲文学高峰，因此演员不仅要会唱，更要能理解曲辞意境与人物情态。 

在表演层面，昆曲综合了唱、念、做、打、舞等多种技艺，动作讲究身段、步法、眼神与水袖配合，整体风格细腻、含蓄、节制，不追求强烈外放，而重视情绪层层渗透、人物气质缓缓铺陈。角色行当较完整，生、旦、净、末、丑等分工清楚，表演程式与音乐结构高度协调。伴奏方面，通常采用笛、箫、笙、琵琶及打击乐等，共同营造清雅而富有层次的声场。正因为这种高度综合性，昆曲长期被视作训练传统戏曲演员的重要基础。 

昆曲的艺术特点还在于“雅”。这种“雅”既表现在文本、音乐和表演上，也表现在整体审美气质上：节奏不急迫，情感不粗放，视觉和听觉都追求工稳、秀雅、蕴藉。它适合表现缠绵深致、富于诗性和人生况味的题材，因此《牡丹亭》《长生殿》《西厢记》等作品能够在昆曲舞台上获得极强感染力。进入当代后，苏州地区通过青春版、旅游演出、博物馆展示、校园普及和戏剧节等多种方式，使昆曲在保留古典规范的同时进入更广泛的传播语境。 

**四、代表作品 / 代表场景 / 代表器物（约220字）**
 昆曲的代表剧目包括《牡丹亭》《长生殿》《西厢记》《浣纱记》等，其中《浣纱记》常被视作昆曲走向成熟戏剧形态的重要标志，《牡丹亭》《长生殿》则是最具传播力和经典性的保留剧目。代表场景既包括专业剧院舞台，也包括苏州园林、古戏台、文化节庆、博物馆社教活动等富有江南气质的演出空间。与其他戏曲一样，昆曲的代表器物包括戏服、盔帽、水袖、折扇以及伴奏乐器等，但其真正核心仍是声腔体系与表演程式本身。今天，青春版《牡丹亭》等作品已成为昆曲当代传播中的标志性文本。 

**五、传承人 / 保护现状 / 当代传播（约230字）**
 苏州是昆曲保护与复兴的重要核心地区之一。2000年以来，苏州持续推进中国昆剧艺术节、虎丘曲会恢复、中国昆曲博物馆建设、苏州昆曲传习所恢复以及地方立法等工作；2006年，《苏州市昆曲保护条例》实施，成为较早专门面向昆曲保护的地方性法规。与此同时，江苏省苏州昆剧院等专业机构在传统折子戏、经典大戏和新编作品演出方面发挥了关键作用。近年来，昆曲通过青春版演出、文旅融合、校园普及、国际巡演、短视频传播等路径不断扩大受众面，尤其在青年群体中的可见度明显提升。苏州的经验表明，昆曲的当代传承并非简单保存，而是在制度保护、学术研究、舞台创作和社会传播之间建立长期联动。 

**六、地域文化关联说明（约180字）**
 昆曲与苏州地域文化的关系极深。它产生于昆山及苏州周边地区，成长于吴地语言、江南音乐、文人审美和城市商业文化共同构成的环境中。苏州官话、江南园林、士人趣味与精工细作的地方文化传统，都深刻影响了昆曲的腔格、辞章和舞台气质。换言之，昆曲之所以呈现出如此典雅、细腻、含蓄的风格，并非孤立的艺术选择，而是苏州文化生态长期沉淀的结果。今天，昆曲仍是理解苏州历史文脉、城市气质与江南审美体系的重要入口。
