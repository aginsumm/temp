#### 非遗类别

曲艺

#### 正文

**一、基本简介（约180字）**
 苏州评弹是苏州评话与苏州弹词的合称，主要以苏州方言演说和演唱，是江南地区最具代表性的曲艺形式之一。它通常由一至数位演员在书场中表演，借助说、噱、弹、唱等手段展开叙事，既有强烈的口头艺术属性，也具有鲜明的音乐性与文学性。苏州评弹发源于苏州地区，后广泛流布于江浙沪等吴语文化圈。其整体艺术风格清新细雅、委婉含蓄、韵味绵长，既能讲述历史公案、才子佳人，也能表现现实题材。作为苏州重要的文化名片，苏州评弹已列入国家级非物质文化遗产代表性项目名录。 

**二、历史渊源（约280字）**
 苏州评弹形成于明末清初，根基在苏州及周边吴语地区的民间讲唱传统。随着江南城镇经济发展、茶馆书场兴盛和市民文化繁荣，评话与弹词逐渐成熟并形成稳定演出空间。苏州评话偏重“说”，以单人叙事、描摹人物、铺陈情节见长；苏州弹词则兼具“说”与“唱”，常由两人搭档，以三弦、琵琶伴奏，艺术层次更为丰富。清代以来，苏州评弹不断向上海、浙江等地传播，形成跨区域的曲艺影响网络。到近现代，书场制度、师承谱系、流派唱腔和经典书目日趋完善，苏州评弹成为江南日常文化消费和城市精神生活的重要组成部分。新中国成立后，专业团体、研究整理和名家创作进一步推动评弹发展，但随着大众娱乐方式转变，其传统书场生态也面临调整与转型。 

**三、核心技艺 / 表演形式 / 工艺特点（约450字）**
 苏州评弹的核心技艺在于“说、噱、弹、唱”。“说”是叙事骨架，要求演员能够清晰交代情节、塑造人物、推进节奏；“噱”是语言机锋与场面调度，既调节气氛，也体现演员口才与现场掌控能力；“弹”和“唱”则使作品超出单纯说书，形成独特的听觉审美。苏州弹词常配以三弦、琵琶，在音乐上形成婉转流丽、细致入微的风格，而评话则更强调语言表演和叙事张力。 

苏州评弹特别依赖方言美感。苏州话本身音韵柔和、声调丰富，非常适合表现细腻情绪和层层递进的叙事节奏。演员在表演中需要通过语气、停连、神态和简洁动作，实现多角色切换和场景转换，让听众在有限舞台条件下“听见人物、看见故事”。这也是评弹作为口头表演艺术的独特魅力：台面不大，但通过语言和音乐即可建立完整的想象空间。 

从艺术气质上看，苏州评弹最突出的特征是“雅”和“细”。它并不依靠强烈外部动作取胜，而重在声音层次、说唱分寸和人物情味。传统书目多涉及历史故事、世情传奇、才子佳人、公案演义等，也逐渐发展出现代题材与短篇新作。当前，苏州评弹在剧场演出之外，也通过非遗展演、博物馆活动、文旅驻演、校园传播和数字平台传播等方式延展影响力，但其真正的核心依然是书场语境中的面对面表达与口传心授。 

**四、代表作品 / 代表场景 / 代表器物（约210字）**
 苏州评弹的代表形态分为苏州评话和苏州弹词两支。传统代表书目涵盖历史演义、公案传奇、才子佳人等类型，演出场景则以茶馆、书场、小剧场和曲艺厅堂最具典型性。与大型戏剧不同，评弹更依赖近距离聆听，因此“书场”本身就是其最重要的文化空间。代表器物主要是三弦、琵琶、醒木、折扇、书桌等，其中三弦与琵琶是苏州弹词最具标识性的伴奏工具。对苏州城市文化而言，评弹不仅是一种表演艺术，也是一种日常生活方式和声音记忆。 

**五、传承人 / 保护现状 / 当代传播（约230字）**
 近年来，苏州持续推动评弹保护与传承。政府部门和行业机构围绕演出场所管理、书场建设、名家培养、资料整理和非遗传播进行系统推进。公开资料显示，苏州评弹仍被视为苏州最具代表性的曲艺艺术之一，并受到社会各界听众关注。当前保护工作的难点在于传统书场数量、年轻观众培育、职业吸引力和稳定演出生态。为应对这些问题，苏州评弹一方面保留经典书目和师承体系，另一方面也积极进入景区、校园、展览、网络短视频和跨界合作空间，以更轻量的传播形式接近年轻群体。其未来关键，不只是“保存节目”，而是让书场文化与当代城市生活重新建立连接。 

**六、地域文化关联说明（约170字）**
 苏州评弹本质上是吴语文化、江南市民生活与苏州城市空间共同孕育的艺术形式。它离不开苏州方言的声音特质，也离不开茶馆书场、街巷生活、文人趣味和精致审美构成的文化土壤。正因为苏州城市气质偏于细腻、雅致、讲求分寸，评弹才形成了慢节奏、重韵味、重语言的艺术风格。可以说，听评弹不仅是在听一个故事，更是在听苏州这座城市的声腔、节奏与情感结构。
