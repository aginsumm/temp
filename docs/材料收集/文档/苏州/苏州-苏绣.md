#### 非遗类别

传统美术

#### 正文

**一、基本简介（约180字）**
 苏绣，即苏州刺绣，是以苏州地区为核心形成并发展起来的传统刺绣技艺，也是中国刺绣的重要代表流派之一，2006年被列入第一批国家级非物质文化遗产名录。它以丝线为主要材料，以绣针为主要工具，在绸、缎、纱、绢等底料上进行艺术表现，兼具实用工艺与审美价值。苏绣整体风格以精细、雅洁、柔丽见长，既能表现花鸟、山水、人物、走兽等传统题材，也不断吸纳现代设计语言，形成兼顾经典性与时代感的工艺体系。 

**二、历史渊源（约280字）**
 苏绣的起源已难确考，但从现存实物和文献可知，至少在宋代以前，苏州地区的刺绣工艺已具相当基础。元代苏绣残片显示，当时已能综合运用多种针法；到明代，苏绣更多吸收文人画稿与江南审美趣味，逐步形成“以绣代画”的艺术取向，风格日益精雅；清代则进一步走向成熟，行业分工与商品生产发达，苏州因此有“绣市”之称。道光年间，丁佩撰成《绣谱》，被视为中国较早的刺绣专业著作之一；清末民初，沈寿创“仿真绣”，推动苏绣由装饰性工艺迈向更高层次的写实表达；20世纪30年代，杨守玉又创“乱针绣”，大大拓宽了苏绣的表现力。新中国成立后，苏州刺绣研究所等专业机构建立，使苏绣在人才培养、针法整理和创作创新方面进入新的发展阶段。 

**三、核心技艺 / 表演形式 / 工艺特点（约470字）**
 苏绣的核心，在于“以针作笔、以线设色”。其工艺流程通常包括立意、选稿、配线、分色、上绷、行针、整理等多个环节，每一步都要求高度经验判断。苏绣非常强调丝线的细分与综合色彩效果，常通过劈丝把一根丝线分成更细的丝缕，以获得更柔和、更精密的层次过渡；再借助不同针法控制丝线走向，使光泽、明暗、肌理与结构同时显现。 

在艺术语言上，苏绣既重“形”，也重“神”。传统苏绣长于表现花鸟鱼虫、江南园林、山水人物，追求线条清润、设色典雅、形象秀逸；成熟作品往往在近看时能见细密针脚，远观时则如绘画一般完整统一。明清以来文人画稿的介入，使苏绣形成了较强的书卷气与画意，这也是它区别于一般民间装饰性刺绣的重要特征。 

从技术谱系看，苏绣不断创新发展。沈寿的“仿真绣”强化了写实、光影与立体感，适合表现人物肖像和动物毛羽；杨守玉的“乱针绣”则打破传统针路秩序，以交错针线塑造油画般的质感与空间感。进入当代后，苏绣在传统平绣、双面绣基础上，继续探索写意绣、融针绣和跨媒介表达，使其既能服务展陈收藏，也能进入服饰、文创、数字传播等新场景。 

**四、代表作品 / 代表场景 / 代表器物（约220字）**
 苏绣的代表性成果，一类体现在经典绣品上，如花鸟册页、人物肖像、动物写生、山水屏风等；另一类体现在双面绣、单面绣、装饰挂屏、团扇、服饰绣品等器物形式中。历史上，苏绣既服务日常生活，也进入宫廷、宅第、礼仪和陈设空间，体现“可用、可赏、可藏”的多重价值。近现代以来，一些高水平苏绣作品多次作为重要展陈作品和文化交流作品出现，题材覆盖山水、花卉、动物与时代主题。当前在中国刺绣艺术馆、苏州博物馆及相关专题展览中，苏绣仍是最能代表苏州工艺美术高度的门类之一。 

**五、传承人 / 保护现状 / 当代传播（约230字）**
 目前，苏绣的核心传承区域主要集中在苏州市区、高新区东渚镇、镇湖街道一带，其中镇湖有“苏绣之乡”之称。国家和地方层面已通过非遗名录、代表性传承人制度、研究机构、展馆平台与产业小镇建设等方式持续推进保护。姚建萍等国家级代表性传承人，在技艺传授、作品创作、国际交流和品牌转化方面发挥了重要作用。与此同时，苏绣也面临传统手工艺普遍存在的人才梯队、市场转型和青年关注度等问题。为应对这些挑战，当代苏绣正通过展览、研培、文旅融合、服饰联名、游戏与数字文创等方式扩大传播面，努力实现从“展柜中的非遗”走向“生活中的非遗”。 

**六、地域文化关联说明（约180字）**
 苏绣与苏州地域文化关系极为紧密。首先，江南水乡长期发达的蚕桑丝织传统，为苏绣提供了物质基础；其次，苏州深厚的园林文化、文人文化和工艺传统，塑造了苏绣清雅、细腻、含蓄的审美气质。可以说，苏绣并非孤立的手工技艺，而是吴地生活方式、审美趣味和经济形态共同作用的产物。今天，镇湖苏绣小镇、中国刺绣艺术馆等空间，又将这种传统进一步转化为产业、旅游、教育和城市文化名片，使苏绣成为理解苏州历史文脉与当代文化形象的重要窗口。
